# CSR Portal

A customer service representative (CSR) portal built with React that allows support staff to manage customer accounts and vehicle subscriptions in a centralized interface.

---

## Features

### Customer Management
- View all registered customers
- Live search by name, email, or phone number
- Detailed customer profile view

### Customer Detail View
- Account information (name, email, phone, etc.)
- Active vehicle subscriptions
- Purchase history in one place

### Account Management
- Inline editing of customer details
- Real-time updates

### Subscription Management
- View active subscriptions per customer
- Add new vehicle subscriptions
- Remove or transfer subscriptions between customers

---

## Tech Stack

- React (Vite)
- React Router v6
- Bootstrap 5
- Firebase
