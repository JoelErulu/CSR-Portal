# CSR Portal

A customer service representative (CSR) portal built with React that allows support staff to manage customer accounts, vehicle subscriptions, and purchase history in a centralized system.

---

## Live Demo

- [CSR Portal](https://csr-portal-two.vercel.app)

---

## Features

### Customer Management
- View all registered customers
- Live search by name, email, or phone number
- Detailed customer profile view

### Customer Detail View
- Account information (name, email, phone, etc.)
- Active vehicle subscriptions
- Purchase history in one place with expandable receipts

### Account Management
- Inline editing of customer details
- Real-time validation and updates

### Subscription Management
- Add vehicle subscriptions per customer
- Assign plans to vehicles
- Remove vehicles from accounts

---

## Tech Stack

- React (Vite)
- React Router v6
- Bootstrap 5
- Firebase (Firestore)
